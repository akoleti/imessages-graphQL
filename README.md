# Realtime Chat App

## Overview
This is a Realtime Chat Application built using the following technologies:

- Next.js: A powerful React framework for building server-rendered applications.
- GraphQL: A flexible query language for APIs, enabling the client to request specific data.
-  Node.js: JavaScript runtime used to run the server-side code.
-  MongoDB: NoSQL database to store user data and chat history.
-  Prisma: Type-safe ORM to interact with the MongoDB database.
-  TypeScript: A strongly typed language that builds on JavaScript, providing static types.

## Features
- Realtime communication between users in chat rooms.
- User authentication and registration.
- Support for multiple chat rooms.
- Persistent chat history stored in MongoDB.
- Optimized server-side rendering (SSR) with Next.js.
- API built with GraphQL to handle data fetching and subscriptions for real-time updates.
- Backend logic powered by Node.js.
- Prisma for easy database management with MongoDB.
- Written in TypeScript to enhance code quality and reduce runtime errors.

## Installation and Setup
Prerequisites
- Node.js: Ensure you have Node.js installed on your machine (version 14.x or higher).
- MongoDB: Set up a MongoDB instance either locally or using MongoDB Atlas.
- Prisma: You'll also need Prisma to manage your database schema.
